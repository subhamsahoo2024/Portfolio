# Portfolio Website

A modern portfolio website built with [Next.js 16](https://nextjs.org/).

## Features

- Fast and optimized performance
- Responsive design
- Mobile-friendly
- SEO-optimized

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone <repository-url>
cd next-app
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
npm start
```
